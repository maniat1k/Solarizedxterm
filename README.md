# Solarized - xterm

This configuration is mande for the xterm the simplest terminal ever!

## Get this done in two simple steps.

1- Copy the content of this ".Xdefaults" to yours (or replace it).

	cp .Xdefaults ~/

2- Execute  xrdb .Xdefaults and that's all.
